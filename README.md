Overview

This experiment introduces fundamental data visualisation techniques using Python. It focuses on basic charts and visual encoding methods to represent data clearly and effectively. Both Matplotlib and Seaborn libraries are used to create and compare visualisations.

Tools and Technologies Used
Python
Pandas
Matplotlib
Seaborn
Objectives
To understand basic chart types
To visualise trends, comparisons, distributions, and relationships
To learn visual encoding techniques such as colour, size, and labels
To compare standard and advanced visualisation libraries
Dataset Description

The dataset represents a student's weekly performance with the following attributes:

Days
Study Hours
Marks
Attendance
Sleep Hours
Assignments Completed
Visualisations Implemented
1. Line Chart
Displays trend of study hours over days
Helps in identifying patterns over time
2. Advanced Line Chart
Compares study hours and marks
Uses multiple lines and legends for clarity
3. Bar Chart
Compares marks across different days
Useful for categorical comparison
4. Advanced Bar Chart
Displays values on top of bars
Includes grid for better readability
5. Histogram
Shows distribution of marks
Useful for frequency analysis
6. Scatter Plot
Displays relationship between study hours and marks
Helps identify correlation
7. Conditional Scatter Plot
Uses colour encoding to represent result (Pass/Fail)
Enhances data interpretation visually
Seaborn Visualisations
1. Line Plot
Cleaner and more aesthetic version of line chart
2. Histogram
Improved visual representation of distribution
3. Scatter Plot
Enhanced scatter plot with better styling
Key Insights
Increased study hours generally lead to better marks
Visual encoding (colour, labels, size) improves understanding
Seaborn provides more visually appealing plots compared to Matplotlib
Basic charts form the foundation for advanced data analysis
Conclusion

This experiment demonstrates how simple charts can effectively communicate data insights. Understanding these basic visualisation techniques is essential before moving to more complex analytical methods. The use of both Matplotlib and Seaborn highlights the importance of choosing appropriate tools for better visual representation.
